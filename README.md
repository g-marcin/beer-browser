# beer-browser

## locally:

1. clone repository.
2. open repository in your IDE ( eg vs code )
3. open terminal( crtl + ` )
4. npm install // ( Node js required )
5. npm start
6. open browser and connect to localhost:5173

## live:

<a> https://beer-browser2.netlify.app/ </a>
